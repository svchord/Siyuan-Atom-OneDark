# Siyuan-Simple-OneDark

## 简介

- 参考
  - VS Code的One Dark Pro主题
  - Typora的OneDark主题
  - Mark Text的One Dark主题
- 所做的工作
  - 整体配色更改
  - 样式更改
    - 大纲
    - 表格
    - 复选框
    - 代码块
    - 引用块
